# 轲影像云相册 - 用户端演示版

## 项目说明

这是一个纯前端演示版，模拟轲影像云相册系统的用户端小程序核心功能，用于向客户展示界面和交互流程。

## 包含页面

1. **首页** - 活动列表（进行中/即将开始/已结束）
2. **活动详情** - 轮播图、人脸识别找自己、号牌搜索
3. **云相册** - 图片/视频浏览、瀑布流/方格图切换、专辑分类
4. **大图预览** - 带水印预览、下载、海报生成
5. **购买结算** - 选择资源、套餐优惠、支付流程
6. **个人中心** - 我的订单、下载记录、消息通知

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 无后端依赖，纯模拟数据

## 部署方式

### GitHub Pages

1. 在 GitHub 创建新仓库（如 `kephoto-demo`）
2. 将本目录所有文件推送到仓库
3. 在仓库 Settings > Pages 中启用 GitHub Pages
4. 选择 main 分支作为源
5. 访问 `https://你的用户名.github.io/kephoto-demo/`

### 本地预览

直接用浏览器打开 `index.html` 即可。

## 数据来源

所有数据均为模拟数据，使用 Unsplash 图片作为示例。
